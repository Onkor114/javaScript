Lesson: Building a Responsive Navigation Bar with a Toggle Menu

Objective:

In this lesson, students will learn how to create a fully functional, responsive navigation bar using HTML, CSS, and JavaScript. The navbar will have a toggle button to handle mobile views and adapt to various screen sizes.

Topics Covered:

HTML structure for a responsive navbar
Styling with CSS (Flexbox, Media Queries)
Adding interactivity with JavaScript (Toggle Button for mobile)
Understanding breakpoints and responsive design

1.1 Understanding the Navbar Elements

The navigation bar consists of:
Brand Title: The website's logo or name.
Navigation Links: Links for various sections of the website.

Part 2: CSS Styling

The CSS file provides styling for the navigation bar, making it responsive for both desktop and mobile devices. Flexbox is used to align the navbar items.

styles.css

CSS
Copy
/* General Styles */ * { box-sizing: border-box; margin: 0; padding: 0; } body { font-family: Arial, sans-serif; } /* Navbar Styling */ .navbar { display: flex; background-color: #333; color: white; justify-content: space-between; align-items: center; padding: 0 20px; } .brand-title { font-size: 1.5em; line-height: 60px; } .navbar-links { height: 100%; } .navbar-links ul { display: flex; list-style: none; margin: 0; padding: 0; } .navbar-links li { padding: 0 10px; } .navbar-links a { color: white; text-decoration: none; font-size: 1em; line-height: 60px; display: block; } .navbar-links a:hover { background-color: #444; } /* Toggle Button */ .toggle-button { position: absolute; top: 15px; right: 20px; display: none; flex-direction: column; justify-content: space-between; width: 30px; height: 21px; cursor: pointer; } .toggle-button .bar { height: 3px; width: 100%; background-color: white; border-radius: 10px; transition: 0.3s; } /* Media Queries for Mobile Screens */ @media (max-width: 768px) { .navbar { flex-direction: column; align-items: flex-start; } .toggle-button { display: flex; } .navbar-links { width: 100%; max-height: 0; overflow: hidden; transition: max-height 0.5s ease-out, opacity 0.5s ease-out; opacity: 0; } .navbar-links ul { flex-direction: column; width: 100%; } .navbar-links li { text-align: center; } .navbar-links a { padding: 10px; line-height: 20px; } .navbar.active .navbar-links { max-height: 300px; opacity: 1; transition: max-height 0.5s ease-in, opacity 0.5s ease-in; } /* Change the toggle button bars for the open state */ .navbar.active .toggle-button .bar:nth-child(2) { opacity: 0; } .navbar.active .toggle-button .bar:nth-child(1) { transform: translateY(8px) rotate(45deg); } .navbar.active .toggle-button .bar:nth-child(3) { transform: translateY(-8px) rotate(-45deg); } }

Part 3: JavaScript for Interactivity

This JavaScript file adds the functionality to toggle the menu when the user clicks on the hamburger button.

script.js

JavaScript
Copy
const toggleButton = document.querySelector('.toggle-button'); const navbar = document.querySelector('.navbar'); toggleButton.addEventListener('click', () => { navbar.classList.toggle('active'); });

Part 4: Explanation

HTML Explanation:

Brand Title: The .brand-title represents the website name.
Navbar Links: The navbar-links contains a list of the navigation links (ul > li), which will collapse and expand depending on the screen size.
Toggle Button: The toggle-button contains three spans that form the "hamburger" icon for mobile view.

CSS Explanation:

Flexbox Layout: The .navbar uses display: flex to align the content. It adjusts to column direction in smaller screens using media queries.
Toggle Button Styling: The toggle button is hidden for larger screens and becomes visible for screens narrower than 768px.
Responsive Behavior: Media queries adjust the navbar layout for mobile screens, collapsing the navigation into a vertical list, and using a "hamburger" menu button to toggle visibility.

JavaScript Explanation:

Toggle Functionality: When the user clicks the toggle button, the navbarâ€™s active class is added or removed, changing the visibility of the navigation links.
